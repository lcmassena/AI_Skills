## Publish Commands

```bash
# Package skill
tar -czvf hexagonal-architect.tar.gz . --exclude='.git' --exclude='*.cs' --exclude='*.csproj'

# Upload to agentskills.io
curl -X POST https://api.agentskills.io/skills \
  -H "Authorization: Bearer TOKEN" \
  -F "file=@hexagonal-architect.tar.gz"
```
